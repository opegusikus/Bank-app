# 大学二年期末グループプロジェクト

ユーザーが支払い、送金、残高確認や取引履歴確認などの基本的なバンキング操作を行えるように設計されたソフトです。全ての処理がクライアントサイドで完結する構造になっています。本来このような処理はサーバー側で行う必要がありますが、課題の仕様上サーバー実装は求められていなかったです。
4人グループ「四銃士」でチームリーダー、テクニカルライター、QAエンジニア、システムアーキテクトがいました。

**セットアップ**
```
git clone https://github.com/opegusikus/Bank-app.git
cd Bank-app/x64/Release
./BankApp.exe
```
